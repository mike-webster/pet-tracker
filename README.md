# Pet Tracker _(pet-tracker)_

Track Pet Activities Throughout the Day

## Install
```
git clone https://github.com/mike-webster/pet-tracker.git
docker-compose up --build
```

#### Dependencies
- Ruby 2.5.5
- Rails
- MySql

## Usage
`docker-compose up --build`

#### Generate Test Data
`rake backfill:populate_test_info`
